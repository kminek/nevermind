# nevermind

Tool to automate boring tasks when creating new Laravel project

## Install

```bash
git clone https://github.com/kminek/nevermind.git
cd nevermind
sudo ./install.sh
```

## Uninstall

```bash
cd nevermind
sudo ./uninstall.sh
```

## Usage

```bash
mkdir todo-app
cd todo-app
nevermind todo-app
```
